# Book Courier

**Purpose:**  
Book Courier is a platform designed to streamline the delivery of books. Users can request book pickups and deliveries, track their orders, and manage their book shipments efficiently. It aims to make book logistics simple, fast, and reliable.


## Key Features
- **Book Pickup & Delivery Requests:** Users can schedule pickups and deliveries for books.  
- **Real-Time Tracking:** Track the status and location of book deliveries.  
- **User Authentication:** Secure login and registration system.  
- **Order History:** Users can view past deliveries and their details.  
- **Responsive Design:** Works seamlessly on mobile, tablet, and desktop.  
- **Admin Dashboard (Optional):** Manage deliveries, users, and track system performance.  

## Technologies & NPM Packages Used
- **React** - Frontend library for building UI components.  
- **Next.js** - Framework for server-side rendering and routing.  
- **Tailwind CSS** - Utility-first CSS framework for styling.  
- **Framer Motion** - Smooth animations and transitions.  
- **Axios** - Handling API requests.  
- **React Icons / Lucide-React** - Icon library for UI elements.  
- **React Router (if using)** - Frontend routing for pages.  
- **Firebase / MongoDB (if using)** - For backend database and authentication.
